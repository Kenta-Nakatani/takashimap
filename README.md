# 高島市思い出マップ

地元住民の皆様の貴重な記憶と体験をデジタル地図上に集約し、地域コミュニティの形成と活性化を図るプロジェクトです。

## プロジェクトの目的

- 高島市住民の思い出を地図に集約
- 世代を超えた記憶の共有と継承
- 地域コミュニティの形成と活性化
- 高島市への愛着と誇りの醸成

## 機能

### 思い出マップ
- 航空写真ベースの地図表示
- 思い出スポットの可視化
- カテゴリ別分類（ROMANCE, NOSTALGIA, FUN, DISCOVERY, ADVENTURE）
- 検索・フィルタリング機能

### データ収集
住民への直接インタビューを通じて以下の情報を収集：
- 思い出の場所
- その場所での体験や思い出の内容
- 体験者のニックネーム（プライバシー保護）

## 技術仕様

- **フロントエンド**: HTML5, CSS3, JavaScript
- **地図**: Mapbox GL JS
- **データ**: Google Sheets API
- **アイコン**: カスタムPNG画像

## ファイル構成

```
├── index.html          # メインページ
├── main.js            # メインロジック
├── style.css          # スタイルシート
├── map.geojson        # 高島市境界データ
├── logo.png           # 高島市ロゴ
├── *.png              # アイコン画像
└── README.md          # このファイル
```

## 使用方法

1. `index.html`をブラウザで開く
2. 検索ボタンで思い出を検索
3. カテゴリで絞り込み検索
4. マーカーをクリックして詳細表示
